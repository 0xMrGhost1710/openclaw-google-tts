# Google TTS Skill for OpenClaw

Free text-to-speech using Google Translate.

## Usage

```bash
/gtts "Hola mundo" --lang es
/gtts "Hello world" --lang en --speed 2
```

## Features

- 100% Free - No API keys
- Multi-language
- MP3 output

## Languages

es, en, fr, de, it, pt, ru, ja, ko, zh-CN
